# PRD — 5월 미국 동부 7일 여행 계획 페이지

## 목적
사용자가 여행 계획의 각 단계(STEP)를 순서대로 탐색하며, 개요 → 도시 → 일정 → 날씨 → 예산 → 체크리스트 → 팁 순으로 정보를 소비할 수 있는 단일 페이지 여행 플래너.

---

## STEP 정의

### STEP 1 — 여행 개요 (Overview)

**목록 뷰에서 보여주는 정보**
- 총 여행 기간: 7일
- 핵심 도시 수: 4개
- Amtrak 이동 횟수: 3회
- 5월 평균 기온: 18°C
- 워싱턴 D.C. 스미소니언 입장료: 무료
- 비자 정보: ESTA 무비자

**클릭 시 열리는 상세 정보**
- 각 stat 카드 클릭 → 인라인 팝오버(tooltip) 또는 하단 확장 패널
  - `7일` 카드 → "2025년 5월 10일(토) 출국 ~ 5월 16일(금) 귀국. 비행 포함 실제 8일 일정."
  - `4개 도시` 카드 → "뉴욕(2박) · 워싱턴 D.C.(2박) · 필라델피아(1박) · 보스턴(귀국)"
  - `Amtrak 3회` 카드 → "① NY→DC 2h 45m ② DC→Philly 2h ③ Philly→Boston 5h 30m"
  - `18°C` 카드 → "뉴욕 14~22°C · D.C. 16~26°C · 필라델피아 14~24°C · 보스턴 12~20°C"
  - `무료` 카드 → "스미소니언 산하 19개 박물관 전부 무료. 국립 항공우주·자연사·역사 박물관 포함."
  - `ESTA` 카드 → "CBP 공식 사이트 신청. 수수료 $21. 출발 최소 72시간 전 필수."
